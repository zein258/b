
# User Management API

A simple Node.js + SQLite API for managing users.

## Setup

```
npm install
npm start
```

## Endpoints

- `POST /signup`
- `GET /users`
- `GET /user/:user_id`
- `GET /user/:user_id/field/:fields`
- `PUT /activate/:user_id`
- `PUT /update/:user_id`
- `PUT /block/:user_id`
- `PUT /unblock/:user_id`
- `DELETE /user/:user_id`
